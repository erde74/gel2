This directory contains a set of benchmark programs to compare GEL2's performance with that of popular object-oriented programming languages.  Each benchmark includes versions written in C++, Java, C# and GEL2.  The benchmarks include the following:
  • binarytrees: from the Computer Language Shootout Benchmarks.  This program allocates and frees a large number of binary trees.
  • nsieve: also from the Computer Language Shootout site.  This program counts prime numbers using the well-known Sieve of Eratosthenes algorithm.  It performs few memory allocations; its inner loop involves arrray accesses.
  • sort: on each of a series of iterations, this program generates a list of 1,000,000 randomly generated integers and sorts them using a recursive mergesort algorithm.
  • sortstring: like the sort benchmark, but uses a list of 400,000 randomly generated strings.
